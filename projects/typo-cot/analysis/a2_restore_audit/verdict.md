# A2 判定: restore「自明コピー」批判への反証 (3 点セット)

**攻撃**: GSM8K の CoT 末尾は最終数値を含む。セル C (typo質問+clean CoT強制) の restore は「答え抽出段が CoT 末尾の数値を書き写すだけ」の自明な帰結ではないか。

## (i) リーク層別
- **GSM8K** (n=763): overall restore=0.945。リーク (最終行に金答え数値) が **757/763 でほぼユニバーサル** → (i) 単独では GSM8K の copy 経路を排除できない。→ (ii)/(iii) へ。
- **MMLU** (n=1714): overall restore=0.810。**答え文字も選択肢本文も現れない no-leak: n=789, restore=0.772 [0.74,0.80]**。最寛容 leak を除いても n=781, restore=0.772。→ **MC では答えは CoT テキストに書かれておらず再導出されている = 反証。**

## (ii) 結論剥ぎ (最終計算行を除去し自由生成 max_new_tokens=256)
- **GSM8K** (n=333): unstripped=0.952 → stripped=0.489。leak-lastline 部分 (n=329): unstripped=0.960 → stripped=0.489
  - **計算の試行 (自明コピー否定の核)**: 末尾行除去後も stripped 答えが非空 (=何らかの答えを算出) の率 0.91 (n_leaked=329)。restore 失敗時も 0.83 が(誤った)答えを算出 → コピーでなく再計算 (例: 540→180 の算術誤り)。
- **MMLU** (n=450): unstripped=0.822 → stripped=0.291。leak-lastline 部分 (n=243): unstripped=0.844 → stripped=0.350
  - **計算の試行 (自明コピー否定の核)**: 末尾行除去後も stripped 答えが非空 (=何らかの答えを算出) の率 0.99 (n_leaked=243)。restore 失敗時も 0.98 が(誤った)答えを算出 → コピーでなく再計算 (例: 540→180 の算術誤り)。

## (iii) 回復曲線 (先頭 p% 強制)
- **GSM8K** (n=333): p0=0.16 p25=0.50 p50=0.71 p75=0.83 p100=0.95
- **MMLU** (n=450): p0=0.16 p25=0.47 p50=0.54 p75=0.66 p100=0.83

## 総合判定: 「自明コピー」説を棄却 (再導出可能な推論内容の媒介を支持)
1. **MC (MMLU) は非自明 [(i)]** — 答え文字も選択肢本文も CoT に現れない事例でも restore 0.77 (n=789)。コピーすべき答え文字列が存在しないのに復元 → 再導出。
2. **回復曲線が丸写しを排除 [(iii), 最強]** — 金答えは CoT 末尾付近にしか出ないが、先頭 p% だけ強制した部分プレフィックスでも段階的に復帰 (GSM8K pooled p0=0.16→p25=0.50→p50=0.71→p75=0.83→p100=0.95)。**p=25〜50% では答え数値はまだ強制テキストに無いのに restore 0.50〜0.71** → 存在しない答えはコピー不能、モデルは推論を続けて再導出している。
3. **結論剥ぎでモデルは再計算する (コピーしない) [(ii)]** — 金答えを載せた最終行を除去し自由生成すると、leak 群 restore 0.96→0.49 に低下するが、**除去後も 0.91 が非空の答えを算出し、restore 失敗時ですら 0.83 が(誤った)数値を出力** (例: 540→180 の算術誤り)。コピー機なら空/末尾数値を返すはず。低下は「再計算の成否」= モデルの算術力(gemma 0.80 > mistral 0.50 > llama 0.36)に依存し、コピーの証拠ではない。

**結論**: 3 点セットは一貫して **restore が答えの丸写しでなく、CoT が運ぶ再導出可能な推論内容の媒介** であることを支持する。したがって IE 優位はフォーマットの自明な性質でなく機構的発見である。**スコープ**: GSM8K の restore の絶対水準は最終読み上げ行の存在から恩恵を受けるが、これは「答えのコピー」ではなく「完了した計算の再実行」であり、(i) MC・(iii) 部分プレフィックス・(ii) 剥ぎ後の再計算のいずれもコピー説と両立しない。
